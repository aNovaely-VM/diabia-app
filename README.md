# 🩺 DIABIA

**Application intelligente pour diabétiques sous pompe à insuline Medtronic MiniMed 780G**

DIABIA transforme les données brutes de ta pompe en conseils clairs et actionnables — comme un professionnel de santé dans ta poche.

---

## 🚀 Fonctionnalités

### ✅ En cours
- **📸 Photo → Glucides** : Prends en photo ton repas et obtiens une estimation précise des glucides

### 🔜 À venir
- **📄 Analyse de rapports** : Upload ton PDF Medtronic → explications en langage simple
- **📊 Courbe glycémique** : Visualisation claire et annotée de tes données
- **💡 Conseils personnalisés** : Ajustement des ratios, basal, objectifs SmartGuard
- **🍽️ Calculateur de bolus** : Basé sur tes ratios personnels

---

## 🏗️ Stack technique

- **Frontend** : React + Tailwind CSS
- **Backend** : Node.js / Deno (Base44 functions)
- **IA Vision** : GPT-4o Vision pour l'estimation des glucides
- **Base de données** : Base44 entities
